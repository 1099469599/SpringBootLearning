# SpringBoot Learning #

    此项目主要作为SpringBoot学习，和根据实际项目对SpringBoot进行配置改造
    主要使用框架：SpringBoot(Spring,SpringMVC),Mybatis,ehcache,javamail。
    项目持续维护中...
